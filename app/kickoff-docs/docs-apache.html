Setting up Apache

Mac OS X comes with a version of the Apache webserver installed. It is suggested that for consistency we all set up our Macs to use this version. (It is possible to set up your Mac to use MAMP or XAMPP stacks and if you have already done so then it's not a big deal).

Setting up the default Apache instance

Follow the instructions given here to set up Apache (NOTE: you don't need to set up PHP and/or mySQL unless you intend developing Wordpress sites). You don't need to worry about the Override .htaccess stuff.
